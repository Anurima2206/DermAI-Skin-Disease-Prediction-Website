# 🩺 DermAI

## 📌1️⃣ About

DermAI is an AI-powered web platform that helps users detect potential skin diseases from uploaded images using image-based disease detection.

Users can upload a skin image and receive a predicted skin disease class generated by a trained deep learning model. The platform also provides personalized treatment advice, downloadable medical reports, and nearby dermatologist recommendations.

This project combines machine learning, backend APIs, and an interactive web interface to create an accessible preliminary skin health assistance tool.

---

# 🚀2️⃣ Features

🔬 **AI Skin Disease Detection**  
Users upload a skin image and receive predictions generated by a trained deep learning model.

📷 **Image Upload & Intelligent Processing**  
Supports common image formats (JPG, JPEG, PNG) with automated preprocessing before prediction.

📍 **Nearby Dermatologist Suggestions**  
The system detects the user’s location and suggests dermatologists within a **5 km radius** using Google Maps services.

🩺 **Personalized Medical Advice**  
Users can enter symptoms and medical details to receive AI tailored treatment suggestions and precautions.

📄 **Downloadable Prescription Report**  
Prediction results and medical advice can be exported as a **PDF report** for future reference.

⚡ **Fast Backend Processing**  
Images are processed through a backend API and analyzed by the ML model in real time.

🌐 **Interactive Web Interface**  
A simple and user-friendly interface allows easy image upload, prediction viewing, and medical guidance.

# ⚙️4️⃣ How It Works

1️⃣ The user uploads a skin image through the web interface.

2️⃣ The image is sent to the backend server.

3️⃣ The backend forwards the image to the **FastAPI ML service**.

4️⃣ The **TensorFlow model analyzes the image** and predicts the skin disease class.

5️⃣ The result is returned to the interface along with medical advice and nearby doctor suggestions.

---

# 🧠 Skin Diseases Detected

The model is trained to classify the following skin conditions:

- Eczema  
- Warts  
- Melanoma  
- Atopic Dermatitis  
- Basal Cell Carcinoma  
- Melanocytic Nevi  
- Benign Keratosis  
- Psoriasis / Lichen Planus  
- Seborrheic Keratosis  
- Tinea / Fungal Infection

## Option 2 — Run Locally

### Clone the repository
```
git clone https://github.com/Anurima2206/Skin-Disease-Prediction-Website.git

cd Skin-Disease-Prediction-Website
```
### Install backend dependencies
```
cd backend

npm install
```
### Run the server
```
npm run server
```
### Run ML API
```
pip install -r requirements.txt

<!-- Provide a Gemini API Key in the ml_api.py file -->

uvicorn ml_api:app --reload
```
### Open the frontend in your browser to start testing.
```
cd frontend
npx serve i -g
npx serve
```
